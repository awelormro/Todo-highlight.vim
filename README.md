# Todo-highlight.vim
 Highlight the TODO labels in vim, with the following settings:

- In green:
  - TODO, DONE, NOTE
- In yellow:
  - WARN, WARNING
- In red the following:
  - FIXME, XXX, BUG, HACK, REPAIR

No configs needed.

# Installation

Using vimplug:

```
Plug 'awelormro/Todo-highlight.vim'
```

Using packer:

```
Plug 'awelormro/Todo-highlight.vim'
```
